## Description

Docker-compose setup for starting [Prometheus](https://prometheus.io).

## Setup

```sh
cd /opt
git clone https://github.com/solution-libre/docker-prometheus.git prometheus
cd prometheus
```

Change the hostname value in `.env`.

## Usage

```sh
cd /opt/prometheus
docker-compose up -d
```
